# youtube-trends

A dashboard for analysis and visualization of Youtube trends

## Requirements

- Docker
- Docker Compose

## Tech Stack

- ReactJS
- Nivo Charts(D3)
- JavaScript
- FastAPI
- Pandas and Numpy
- Python
- MongoDB
- Redis Cache

## Services

- CLI (ETL TOOL)
- Client
- Server
- Notebook

## Instructions

Start the instances

```bash
$ docker-compose up --build
```

Visit [http://localhost:3000](http://localhost:3000) in your local browser.
